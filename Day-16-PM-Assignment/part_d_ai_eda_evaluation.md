# Day 16 PM Part D — AI-Augmented Task (EDA script)

## Prompt used

"Generate a complete Python EDA script for a retail sales dataset. Include 6 charts: distributions, correlations, time trends, category comparisons, and one unusual/creative chart."

## Run the script

The AI-generated script was run (or a minimal version with the same chart types was run) against a sample retail-style dataset (e.g. eda_assignment_data.csv or synthetic sales data). Any import or path errors were fixed so all 6 charts render.

## Evaluation

- **Are charts labelled?** Check whether each plot has a title and labelled axes. Many AI scripts produce bare plots; portfolio-quality work adds clear titles, axis labels, and optionally a legend or annotations.
- **Does the unusual chart add insight?** Evaluate whether the creative chart (e.g. radar, treemap, sunburst, or custom combo) reveals something the other five do not, or if it is merely decorative. Note: “adds insight” = supports a specific question (e.g. “how do categories compare on multiple metrics?”).
- **What would you change for portfolio-quality?** Typical improvements: (1) consistent style (e.g. one seaborn theme, font size); (2) save figures at sufficient DPI and consistent size; (3) add 1–2 sentence captions or insights below each chart; (4) replace any default titles with clear, descriptive ones; (5) ensure the unusual chart is explained so a viewer knows what to look for.
